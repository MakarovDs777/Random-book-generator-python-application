# Random book generator 1

Сделал генератор случайных книг хотя в интернете есть куча сайтов где генерируются случайные книги с осмысленным текстом но приложения на python ещё никто не делал как мне пришла идея разделить по умолчанию 10 на 10 столбцов, и в каждом столбце сделать случайную букву включая запятые, и пробелы, и за определенное время по умолчанию 0.6 секунды каждая буква меняется на случайную типа как в матрице если кто помни [такой screensaver ввиде дождя из букв](https://www.youtube.com/watch?v=1e-8ydjiAg8).
